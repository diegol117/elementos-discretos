# Estructuras Discretas — Recursos de Matemáticas Discretas 📚

¡Bienvenido al repositorio oficial de recursos académicos para la asignatura **Estructuras Discretas (EDI-S8)**! Este espacio ha sido diseñado para centralizar material de apoyo, guías y ejercicios prácticos para estudiantes y entusiastas de la lógica y la matemática discreta.

🌐 **Accede al sitio web aquí:** [https://diegol117.github.io/elementos-discretos/](https://diegol117.github.io/elementos-discretos/)

---

## 🎯 Objetivo del Proyecto
Proporcionar una plataforma accesible y organizada con recursos de alta calidad sobre los fundamentos de las estructuras discretas, facilitando el aprendizaje autodidacta y el repaso de conceptos clave vistos en clase.

## 🗂️ Contenido del Repositorio
En este sitio encontrarás material detallado sobre:

* **Lógica Matemática:** Tablas de verdad, cálculo proposicional y de predicados.
* **Teoría de Conjuntos:** Operaciones, diagramas de Venn y propiedades.
* **Relaciones y Funciones:** Relaciones de equivalencia, orden parcial y diagramas de Hasse.
* **Estructuras Algebraicas:** (Próximamente) Grupos, anillos y cuerpos.
* **Grafos y Árboles:** Fundamentos y aplicaciones.

## 🚀 Tecnologías Utilizadas
Este recurso ha sido construido utilizando:
* **HTML5/CSS3** para la estructura y el diseño responsivo.
* **JavaScript** para componentes interactivos.
* **LaTeX** para la representación precisa de fórmulas matemáticas.
* **GitHub Pages** para el despliegue y hosting gratuito.
